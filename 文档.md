# Documentation

## 布局

### eg-layout

- 用法

```html
<div class="eg-layout">
    <div class="eg-header">header</div>
    <div class="eg-nav">nav</div>
    <div class="eg-content">content</div>
    <div class="eg-footer">footer</div>
</div>
```

- 渲染结果

```
------------------------
|   header             |
|                      |
------------------------
|   nav |   content    |
|       |              |
------------------------
|   footer             |
|                      |
------------------------

```
